# ariaSoftware - Gestión de Clientes

Este es un sistema de gestión de clientes (CRUD) desarrollado con **PHP** y **MySQL** utilizando la interfaz de **PDO** para garantizar seguridad y eficiencia.

## 🚀 Características
- Registro, edición, visualización y eliminación de clientes.
- Sistema de ubicación dinámica por Departamentos y Ciudades de Colombia.
- Búsqueda de clientes por nombre o documento.
- Interfaz limpia y responsive con CSS personalizado.

## 🛠️ Tecnologías utilizadas
- **Lenguaje:** PHP 8.x
- **Base de Datos:** MySQL / MariaDB
- **Estilos:** CSS3
- **Scripting:** JavaScript (para recarga dinámica de formularios)

## 📋 Instalación
1. Clona este repositorio en tu carpeta `htdocs` de XAMPP.
2. Crea una base de datos llamada `crud` en phpMyAdmin.
3. Importa el archivo situado en `/db/crud.sql`.
4. Configura tus credenciales en el archivo `conexion.php`.
5. Accede desde tu navegador a `http://localhost/taller/index.php`.
